# Learning_WebApi

> * _練習存取後端Api中的json資料

* 使用工具
  1. Visual Studio 2019
  2. Sourcetree 3.3.9
  3. Postman

* 說明
  1. 建立`ASP.NET MVC Web API`專案
  2. 建立`LocalDB`，輸入測試資料
  3. 分別以`ViewBag`、`Model`、`$.get`、`$.ajax`取得後端資料
  4. 前端以`@Json.Encode()`、`JSON.stringify()`等方法將物件解碼json資料
  5. 使用`Postman`可快速測試API
